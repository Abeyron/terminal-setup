## ⚡ Super Quick Start

**Windows (5 minutes):**
```powershell
cd terminal-setup\scripts
powershell -ExecutionPolicy Bypass -File install.ps1
# Restart Windows Terminal
```

**Linux/WSL (3 minutes):**
```bash
cd terminal-setup/scripts
./install-linux.sh
source ~/.bashrc
```
